  • html+css character builder

This project was mostly about interacting with the DOM.

Selecting a class will change the background and add additional text to the screen.

Selecting a race will give additional points to specific stats and reset stats if you change race.

Clicking a stat will utilize a prompt so the user and enter how much they'd like to change.

Changing a stat will update the button face and remaining points accordingly.

Trying to use more than the allotted points will do nothing.
